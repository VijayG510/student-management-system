# Student Management System

## Description
This project is designed to manage student information such as
student ID, name, age, and department. It supports basic CRUD
operations for maintaining student records.

## Technology Used
- Core Java (Console Application)
OR
- SQL (Database based project)  ← one option மட்டும் வைத்துக்கலாம்

## Features
- Add new student
- View all students
- Search student by ID
- Delete student record

## Concepts Used
- Classes and Objects
- Encapsulation
- ArrayList / SQL Tables
- Conditional statements and loops
